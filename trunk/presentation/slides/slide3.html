Javascript

  • Hovedsyntaks som kjent fra Java, men 100% dynamisk.
  • Har objekter. Disse skiller seg noe ut:
    • Et objekt defineres som en funksjon.
    • I funksjonen er this objektet selv. Alle funskjoner og variable akssesseres via this.
    • Funksjoner defineres som variable på objektet.
      this.foo = function(param1,param2) { ... }
    • Attributter defineres når de trenges - this.bar = 42;
  • Mye funksjonalitet for å interaktere med DOM - det er utenfor scopet av denne workshoppen. Vi har brukt Prototype for å gjøre dette enklere i koden vår.
