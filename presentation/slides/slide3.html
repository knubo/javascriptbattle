Javascript

  • Hovedsyntaks som kjent fra Java, men 100% dynamisk og utypet.
  • Har objekter. Disse skiller seg noe ut:
    • Et objekt defineres som en funksjon.
    • I funksjonen er this objektet selv. Alle objektets funskjoner og variable aksesseres via this.
    • Funksjoner defineres som variable på objektet.
      this.foo = function(param1,param2) { ... }
    • Attributter defineres når de trenges: this.bar = 42;
    • Alt er public. Men variable har scope til objektet (kodeblokken).
